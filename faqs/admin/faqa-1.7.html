Sending Change Password Links: [Updated as of May-2024]

Safeguard Account Security by Enabling Employees to Easily Update and Manage Their Passwords for Enhanced Data Protection.


Sending change password links is a vital aspect of safeguarding account security and enhancing data protection measures within the HRMS system.
Here's a detailed guide on how to send change password links:


1. Sending Change Password Links:
• Navigate to the Employee Management Tab: From the sidebar menu on the dashboard, access the employee management tab.
• Employee Overview: In the employee management section, locate the table displaying employee details. Click on the "View More" button associated with the specific employee requiring a password change.
• Initiating Password Change Request: On the employee overview tab, locate the "Requests" button at the top left corner. Hover over the button to reveal a dropdown menu.
• Selecting Change Password Option: From the dropdown menu, choose the "Change Password" option. This action triggers the system to generate a change password link for the employee.
• Sending the Link: Once the "Change Password" option is selected, the system automatically generates a unique link for the employee to change their login password.
• Employee Action: The generated link is then sent to the respective employee via email. The employee can click on the link to navigate to the password reset page.
• Password Update: Upon clicking the link, the employee will be directed to a secure page where they can update their login password as per the organization's password policy.
• Confirmation: After the password is successfully updated, the employee receives confirmation of the password change, ensuring enhanced account security.


By following these steps, organizations can enable employees to easily update and manage their passwords, thereby strengthening account security and enhancing data protection measures within the HRMS system.
